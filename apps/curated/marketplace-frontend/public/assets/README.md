# Assets Directory

## Homepage Assets Needed

### Hero Section

#### Video Background (Recommended)
- **Files**:
  - `hero-video.mp4` - Primary video format
  - `hero-video.webm` - Alternative format for better browser support
- **Dimensions**: 1920x1080 (Full HD) or 3840x2160 (4K)
- **Duration**: 10-30 seconds (will loop automatically)
- **Format**: MP4 (H.264) and WebM (VP9)
- **Description**: Looping video of spa treatments, skincare products, or serene spa environment
- **Requirements**:
  - No audio required (video plays muted)
  - Smooth loop point (end should transition seamlessly to beginning)
  - Optimized file size (recommended: 5-15MB for MP4, 3-10MB for WebM)
  - Subtle, non-distracting motion

#### Fallback Image (Required)
- **File**: `hero-background.jpg`
- **Dimensions**: 1920x1080 (or larger)
- **Format**: JPG or WebP
- **Description**: High-quality spa/skincare background image (shows while video loads or if video fails)
- **Use**: Also serves as video poster frame

### Brand Logos
Location: `brands/`
- `brand1.svg` through `brand8.svg`
- **Format**: SVG (preferred) or PNG with transparent background
- **Dimensions**: 128x64 (or 2:1 aspect ratio)
- **Description**: Featured spa brand logos

### Editorial Content
Location: `editorial/`
- **File**: `curated-collection.jpg`
- **Dimensions**: 800x600 (4:3 aspect ratio)
- **Format**: JPG or WebP
- **Description**: Promotional/editorial image for content block

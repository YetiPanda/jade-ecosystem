# Curated by Jade

B2B Skincare Marketplace applications.

## Applications

### Marketplace Frontend
Consumer-facing marketplace for spa owners to discover and order premium skincare products.

[View Documentation →](./marketplace-frontend/README.md)

### Vendor Portal
Seller dashboard for vendors to manage products, orders, and analytics.

[View Documentation →](./vendor-portal/README.md)

### Vendure Backend
E-commerce backend powered by Vendure framework.

[View Documentation →](./vendure-backend/README.md)

## Development

```bash
# Start all Curated apps
pnpm dev --filter="curated-*"

# Start specific app
pnpm dev --filter=curated-marketplace-frontend
```

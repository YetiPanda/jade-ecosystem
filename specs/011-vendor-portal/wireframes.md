## Design Principles

### Overall Aesthetic
- **Jade green** (#2d5a47) as primary brand color
- **Gold accent** (#b8926a) for CTAs and highlights
- **Clean, professional** appearance suitable for B2B
- **Generous whitespace** - premium marketplace feel
- **Georgia/serif** for headings, **Inter/sans-serif** for body

### Responsive Strategy
- **Desktop-first** design (1440px reference)
- **Tablet breakpoint**: 768px
- **Mobile breakpoint**: 375px
- **Critical flows** must work on mobile (orders, messages, dashboard overview)
- **Complex tables** collapse to cards on mobile

- All interactive elements keyboard accessible
- Color contrast minimum 4.5:1 for text
- Status indicators not color-only (icons + text)
- Form fields with visible labels
- Error messages associated with fields
- Loading states announced to screen readers
